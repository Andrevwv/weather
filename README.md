# Weather app

### Try app [HERE](https://andrevwv.github.io/weather/)

Used:

1. Vue.js
2. Vuex
3. Vue Router
4. axios HTTP client
5. grid & flex layouts
6. transitions on page change

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Deploy to gh-pages

```
bash deploy.sh
```
